# KitaF_Q-Tracker
待ち時間をリアルタイムで確認できるWebアプリ

# 使用技術
現在、使用する予定の技術は下記のとおりです。
- Docker (コンテナ仮想化プラットフォーム)
- Docker compose (コンテナ管理ツール)
- Golang (バックエンド)
- Next.js (フロントエンド)
- Daisy UI (CSSフレームワーク)
- Bootstrap (CSSフレームワーク)
- Sqlite (データベース)
